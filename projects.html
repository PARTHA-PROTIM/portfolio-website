Cadillacs and Dinosaurs Game – Project Summary

Goal:

To recreate the classic arcade game using C++ with SFML, focusing on authentic gameplay, visuals, and a responsive user experience.

Key Features:

  • Data Structures & Algorithms: Arrays, vectors, linked lists, queues, strings.
  • Graphics Rendering: Windows API and SFML for textures, sounds, and UI.
  • Algorithms: Mersenne Twister, linked list traversal, SFML utilities, sorting/searching algorithms.

Excluded Features:

  • Dinosaurs as Enemies: Removed to maintain focus and thematic clarity.
  • Multiplayer Support: Only single-player mode included to avoid complexity.
  • Weapon Selection: Grenades, rifles, and custom weapons not included for simplicity.

Rationale:

These exclusions helped streamline development and focus on core mechanics, ensuring a clean, efficient single-player experience.
